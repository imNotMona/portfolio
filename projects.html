Explicit Free-List Memory Allocator

C · Systems Programming · Memory Management

I implemented a custom dynamic memory allocator in C using an explicit free-list design. The allocator manages a manually mapped heap using mmap and maintains separate doubly linked lists for available and allocated memory blocks.

This project deepened my understanding of low-level memory layout, pointer arithmetic, alignment, and the trade-offs involved in allocator design.
